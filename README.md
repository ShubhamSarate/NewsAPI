# 📰 NewsApp (React)

A simple and responsive news application built with React that fetches and displays top headlines using a news API. The app supports category-based browsing, infinite scrolling, and a loading progress bar.

## 🚀 Features

- Category-wise news browsing (General, Business, Sports, Technology, Health, Science, Entertainment)
- Infinite scrolling for seamless loading of articles
- Top loading progress bar for better UX
- Responsive design
- API integration for real-time news
- Fallback handling for broken images

## 🛠️ Tech Stack

- React.js
- React Router
- Bootstrap
- NewsAPI
- React Infinite Scroll Component
- React Top Loading Bar
